Electron
Cellular connected Internet of Things development kit wit a small footprint powered by a powerful STM32 ARM Cortex M3 microcontroller and a uBlox cellular chip.
Datasheet v005
Features
  • Microcontroller specs
    • STM32F205 120Mhz ARM Cortex M3
    • 1MB Flash
    • 128KB RAM
    • 30 mixed-signal GPIO (digital, ADC, PWM)
    • Advanced peripherals: DAC, I2S, I2C, CAN
  • Cellular specs
    • U-Blox SARA U260/U270 (3G) or G350 (2G)
    • Telefonica global SIM card
    • Particle IoT data plan - Plans vary based on usage and location. See website for details.
  • Other specs
    • On-board RGB status LED
    • Li-Po battery (2000mAh)
    • Board dimensions: 2.0" x 0.8" x 0.3" (0.5" including headers)
    • Breadboard friendly 38-pin package
    • Real-time operating system (FreeRTOS)
    • FCC/CE/IC certified
    • Open source hardware
  • Simple to setup and use
    • Set up with web app within minutes
    • Connects to the Particle cloud out of the box
    • Program over the air through the Particle Web IDE
    • Work on projects locally with the Particle Desktop IDE
    • Script common tasks with the Particle Command Line Interface
    • Send data to 3rd party services through webhooks and integrations
    • Monitor state of device through the Particle Console
  • Community resources
    • Explore fully developed projects to get through IoT learning curve
    • Use community-provided code libraries for common peripherals
    • Join the Particle community for inspiration and help by experts
